حول الأداة

ما هي أداة CyberDev؟

CyberDev هي أداة فحص أمني مفتوحة المصدر مكتوبة بلغة Python، تهدف إلى مساعدة المختصين والطلاب في فحص مواقع الويب واكتشاف نقاط الضعف بطريقة آمنة وقانونية — دون أي استغلال.

تتميز الأداة بواجهة CLI تفاعلية سهلة الاستخدام، وإمكانية تصدير التقارير بعدة صيغ، بالإضافة إلى نظام جلسات لحفظ نتائج الفحوصات السابقة.

طلاب الأمن السيبراني

أداة تعليمية مثالية لفهم نقاط الضعف الشائعة وكيفية اكتشافها في بيئة آمنة.

المختبرات التعليمية

مناسبة للاستخدام في معامل الجامعات ومراكز التدريب لتعليم اختبار الاختراق.

الشركات الصغيرة

فحص سريع لمواقع الشركات الصغيرة والمتوسطة لاكتشاف المشاكل الأمنية الأساسية.

المطورين

أداة سهلة الدمج في عمليات التطوير للتحقق من أمان المواقع قبل الإطلاق.

🛡️

فلسفة الأداة

فحص نقاط الضعف فقط — لا استغلال. CyberDev مصممة لتكون أداة كشف واستطلاع أمني، تساعدك على تحديد المشاكل الأمنية المحتملة دون محاولة استغلالها أو إلحاق أي ضرر.

المميزات

مميزات قوية لفحص شامل

تقدم CyberDev مجموعة متنوعة من الفحوصات الأمنية مع واجهة تفاعلية سهلة الاستخدام.

Misconfigurations

اكتشاف الإعدادات الخاطئة في السيرفر والتطبيق التي قد تؤدي لثغرات أمنية.

Outdated Software

الكشف عن البرمجيات والمكتبات القديمة التي تحتوي على ثغرات معروفة.

Security Headers

التحقق من وجود وتكوين هيدرز الأمان مثل CSP و X-Frame-Options وغيرها.

Email Security (SPF/DMARC)

فحص سجلات DNS للتحقق من حماية البريد الإلكتروني ضد الانتحال.

SSL/TLS Analysis

تحليل شهادات SSL وبروتوكولات التشفير المستخدمة.

Server Information

جمع معلومات السيرفر مثل نوعه وإصداره والتقنيات المستخدمة.

Cookie Security

فحص إعدادات الكوكيز والتحقق من خصائص الأمان مثل HttpOnly و Secure.

Robots.txt Analysis

تحليل ملف robots.txt للكشف عن مسارات حساسة قد تكون مكشوفة.

Port Scanning

فحص المنافذ المفتوحة الشائعة على الخادم المستهدف.

واجهة تفاعلية كاملة

واجهة CLI تفاعلية مع قوائم ملونة وتنظيم واضح للخيارات باستخدام Rich و InquirerPy.

ألوان ورموز واضحة

نتائج ملونة بألوان مميزة لتسهيل التمييز بين النتائج الإيجابية والتحذيرات والأخطاء.

خيارات مرنة

إمكانية تخصيص الفحص واختيار أنواع محددة مع دعم الأوامر المتقدمة.

تصدير JSON

تقارير بصيغة JSON مناسبة للمعالجة البرمجية والدمج مع أدوات أخرى.

تصدير HTML

تقارير HTML جميلة وتفاعلية يمكن مشاركتها مع الفريق أو العميل.

تصدير CSV

تقارير CSV لتحليل البيانات في Excel أو أي أداة تحليل بيانات.

حفظ الجلسات

حفظ نتائج كل فحص تلقائيًا لمراجعتها لاحقًا ومقارنتها بالفحوصات السابقة.

استعراض النتائج

إمكانية استعراض جميع الفحوصات السابقة مع تفاصيل كل فحص.

تتبع التغييرات

مقارنة النتائج عبر الزمن لمعرفة ما إذا تم إصلاح الثغرات.

التثبيت

تأكد من توفر Python 3.8+ على جهازك قبل البدء.

# استنساخ المشروع $ git clone https://github.com/ahmedalhomili/CyberDev.git $ cd CyberDev # إنشاء بيئة افتراضية (اختياري لكن مُوصى) $ python -m venv venv $ source venv/bin/activate # Linux/Mac $ venv\Scripts\activate # Windows # تثبيت المتطلبات $ pip install -r requirements.txt

المتطلبات:

  • Python 3.8 أو أحدث
  • pip (مدير حزم Python)
  • اتصال إنترنت للفحص

البدء السريع

ابدأ أول فحص لك في أقل من دقيقة:

# تشغيل الأداة $ python cyberdev.py # أو مع تحديد الهدف مباشرة $ python cyberdev.py --target https://example.com # فحص محدد $ python cyberdev.py --target https://example.com --scan headers

أوامر CLI

# عرض المساعدة $ python cyberdev.py --help # الأوامر المتاحة: --target, -t تحديد الموقع المستهدف --scan, -s نوع الفحص (headers, email, ssl, all) --output, -o مسار حفظ التقرير --format, -f صيغة التقرير (json, html, csv) --session استعراض الجلسات السابقة --verbose, -v عرض تفاصيل إضافية --version عرض رقم الإصدار

الإعدادات

يمكنك تخصيص إعدادات الأداة عبر ملف config.py:

# config.py TIMEOUT = 30 # مهلة الاتصال بالثواني USER_AGENT = "CyberDev/1.0" MAX_RETRIES = 3 # عدد المحاولات REPORT_DIR = "reports/" # مجلد التقارير SESSION_DIR = "sessions/"

إدارة الجلسات

# عرض جميع الجلسات المحفوظة $ python cyberdev.py --session list # عرض تفاصيل جلسة محددة $ python cyberdev.py --session view --id 1 # حذف جلسة $ python cyberdev.py --session delete --id 1

تصدير التقارير

# تصدير بصيغة JSON $ python cyberdev.py -t https://example.com -f json # تصدير بصيغة HTML $ python cyberdev.py -t https://example.com -f html # تصدير بصيغة CSV $ python cyberdev.py -t https://example.com -f csv # تحديد مسار مخصص $ python cyberdev.py -t https://example.com -f json -o ~/my-reports/

التقارير

نتائج فحص احترافية

تقارير واضحة ومنظمة تساعدك على فهم نقاط الضعف واتخاذ الإجراءات المناسبة.

مثال على تقرير فحص

JSON HTML CSV
الفحص النتيجة الخطورة الحالة
X-Frame-Options غير موجود — الموقع عرضة لـ Clickjacking عالي ضعيف
Content-Security-Policy غير مُعرّف — لا حماية ضد XSS حرج ضعيف
Strict-Transport-Security HSTS مفعّل بشكل صحيح معلومة آمن
SPF Record سجل SPF موجود ومُعرّف بشكل صحيح معلومة آمن
DMARC Record غير مُعرّف — البريد عرضة للانتحال متوسط ضعيف
SSL Certificate الشهادة صالحة — تنتهي بعد 89 يومًا منخفض آمن
Cookie Security بعض الكوكيز بدون خاصية HttpOnly متوسط ضعيف

الاستخدام القانوني

إخلاء المسؤولية والاستخدام الأخلاقي

CyberDev أداة تعليمية — استخدمها بمسؤولية وبشكل قانوني فقط.

الاستخدام المسموح

  • فحص المواقع التي تملكها أو لديك إذن كتابي لفحصها
  • استخدام الأداة في بيئات تعليمية ومختبرات تدريب
  • فحص بيئات الاختبار المحلية (localhost)
  • المساهمة في تطوير الأداة وتحسينها
  • استخدامها ضمن برامج Bug Bounty المصرح بها

الاستخدام المحظور

  • فحص مواقع بدون إذن صريح من مالكها
  • استخدام النتائج لأغراض خبيثة أو ضارة
  • محاولة استغلال الثغرات المكتشفة
  • استخدامها في أنشطة غير قانونية بأي شكل
  • بيع أو مشاركة نتائج الفحص لأطراف غير مصرح لها

تحذير: المطور غير مسؤول عن أي استخدام غير قانوني أو غير أخلاقي لهذه الأداة. المستخدم يتحمل كامل المسؤولية عن كيفية استخدام الأداة والامتثال للقوانين المحلية والدولية.
